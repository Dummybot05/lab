AIM:

	To write a program to user's first and last name and print them in reverse order with a space between them .

PROGRAM:
		
puts "Input your first name:”
fname = gets. Chomp
puts "Input your last name: "
lname = gets. Chomp
puts "Hello #{lname} #{fname}"

INPUT & OUTPUT:

Input your first name: narasimha reddy 
Input your last name: engineering college
 Hello engineering college narasimha reddy

RESULT:
	Thus the program to user's first and last name and print them in reverse order with a space between them.

NOTE: gets is your user's input. chomp is the method to remove trailing new line character i.e. '\n' from the the string. whenever "gets" is use to take i/p from user it appends new line character i.e.'\n' in the end of the string.So to remove '\n' from the string 'chomp' is used.
