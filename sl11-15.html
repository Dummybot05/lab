11. WRITE A TCL SCRIPT TO FIND THE FACTORIAL OF A NUMBER

AIM:

	To write a program to find the factorial of a number.

PROGRAM:
 
set i 1; set fact 1
set x 5
while {$i <= $x} {
   	 set fact [expr $fact * $i]
   	 incr i
}
puts $fact
INPUT & OUTPUT:
Factorial 5
120

RESULT:
	Thus the program to find the factorial of a number.

13. WRITE A TCL SCRIPT FOR SORTING A LIST USING A COMPARISON FUNCTION

AIM:
	To write a program to Sorting a list using a comparison function.

PROGRAM:
proc my compare {arg1 arg2} {
   if {[regexp {test(\d+)} $arg1 -> n1] && [regexp {test(\d+)} $arg2 -> n2]} 
{
       return [expr {$n1 - $n2}]
   }
   return [string compare $arg1 $arg2]
}

set yourlist {test2 test3 test1 test11}
puts [lsort -command mycompare $yourlist]

INPUT & OUTPUT:
$tclsh main.tcl

test1 test2 test3 test11

RESULT:
	Thus the program to Sorting a list using a comparison functions.

15. WRITE A TCL SCRIPT TO COMPARING THE FILE MODIFIED TIMES.

AIM:
	Write a program to comparing the file modified times.

PROGRAM:

proc newer { file1 file2 } {
	if ![file exists $file2] {
		return 1
	} else {
		# Assume file1 exists
		expr [file mtime $file1] > [file mtime $file2]
	}
}


INPUT & OUTPUT:
