20. WRITE A PERL SCRIPT TO PRINT THE FILE IN REVERSE ORDER USING COMMAND LINE ARGUMENTS.

AIM:
	To write a program to print the file in reverse order using command line arguments.

PROGRAM:
	#!/bin/bash
	text=$(cat $1)
	arr=($text)
	al=${#arr[@]}
	let al="al-1"

	while (($al >= 0))
	do
   	 echo -n "${arr[al]}"
   	 echo -n " "
    	let al="al - 1"
	done
	echo
INPUT & OUTPUT:
	 Input is 
	a
	b
	c
	d
	Output is 
	 d
	c
	b
	a	
RESULT:
	Thus the program to print the file in reverse order using command line arguments
